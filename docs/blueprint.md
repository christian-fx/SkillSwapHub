# **App Name**: SkillSwap Hub

## Core Features:

- User Profile Creation: Users can create detailed profiles, listing their skills offered and skills needed.
- Skill Browsing: Browse available skills offered by other users, with filtering and search options.
- Skill Swap Proposal: Propose skill swaps to other users, specifying the skills to be exchanged.
- Direct Messaging: Communicate directly with other users to discuss and coordinate skill swaps.
- Skill Match Tool: Suggest potential skill swaps to users based on their listed skills offered and needed, using a tool that makes logical deductions about good partnerships.

## Style Guidelines:

- Primary color: Earthy green (#8FBC8F) to create a friendly and inviting atmosphere.
- Background color: Light beige (#F5F5DC), desaturated hue similar to the green, providing a warm, neutral background.
- Accent color: Muted brown (#A0522D), an analogous hue to the green that complements the earthy tones, used for highlighting key interactive elements and creating contrast against the lighter background.
- Body and headline font: 'PT Sans' for a modern yet warm feel suitable for all text.
- Use simple, clean icons to represent skills and actions, ensuring clarity and ease of understanding.
- Employ a clean, intuitive layout with clear sections for profiles, skill listings, and messaging.
- Incorporate subtle animations for skill swap proposals and message notifications to enhance user engagement.
